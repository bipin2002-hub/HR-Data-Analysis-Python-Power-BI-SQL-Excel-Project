# HR-Data-Analysis-Python-Power-BI-SQL-Excel-Project
📊 HR Analytics – Employee Attrition Analysis
🔎 Project Overview

This project analyzes employee attrition patterns and workforce demographics using a dataset of 1,470 employees. The primary objective is to identify key factors contributing to employee turnover and provide data-driven insights to support HR decision-making and retention strategies.

📁 Dataset Summary

Total Records: 1,470

Total Features: 41

Missing Values: None

Key Attributes:

Attrition, Department, Job Role, Age, Gender, Marital Status, Education Field, Job Satisfaction, Monthly Income, Work Life Balance, Years at Company, Performance Rating, and more.

🛠️ Tools & Technologies Used

Python (Pandas, NumPy, Matplotlib, Seaborn) – Data Cleaning & EDA

SQL (MySQL) – Data Querying & KPI Extraction

Excel – Initial Data Review & Dashboarding

Power BI – Interactive HR Analytics Dashboard

📌 Key KPIs

Total Employees: 1,470

Active Employees: 1,233

Attrition Count: 237

Attrition Rate: 16.12%

Average Age: 37 Years

Average Job Satisfaction: 2.73 / 4
